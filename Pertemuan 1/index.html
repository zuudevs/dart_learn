Setup, Sintaks, & Tipe Data

Langkah pertama kamu menjadi Dart Developer yang handal (kalau belajarnya bener).

1. Setup Lingkungan & VS Code

Langkah pertama adalah menginstal Dart SDK dan menyiapkan editor kode. Jangan pakai Notepad ya, please. Kita pakai Visual Studio Code (VS Code).

  • Instal Dart SDK: Unduh dari situs resmi Dart sesuai OS kamu.
  • Instal VS Code: Editor sejuta umat, ringan dan powerful.
  • Ekstensi Dart: Wajib install extension Dart di VS Code biar coding-nya enak (ada syntax highlighting & debugging).

2. Struktur Program: Fungsi main()

Setiap aplikasi Dart butuh pintu masuk. Pintu itu namanya fungsi main(). Tanpa ini, kodinganmu nggak bakal jalan.

main.dart 
// Ini adalah titik masuk program Dart
void main() {
  // Kode kita akan ditulis di sini
  print('Hello, World!');
}

Fungsi print() itu buat nampilin teks ke konsol. Sederhana kan?

4. Variabel: var, final, const

var

Otomatis nebak tipe data. Sekali jadi String, selamanya String.

final

Cuma bisa diisi sekali. Kayak janji suci pernikahan.

const

Konstan sejak kompilasi. Nilainya harus fix sebelum program jalan.

variable.dart 
void main() {
  // var: Tipe data 'nama' otomatis jadi String
  var nama = 'Rara';
  
  // final: Gak bisa diubah lagi
  final String pekerjaan = 'Programmer';
  // pekerjaan = 'Chef'; // Error!

  // const: Nilai tetap abadi
  const double pi = 3.14;
}

5. Tipe Data Dasar

Dart itu Strong Typed, jadi dia galak soal tipe data. Ini yang sering dipakai:

  • int : Bilangan bulat (1, 2, 100)
  • double : Desimal (3.14, 2.5)
  • String : Teks ('Halo')
  • bool : true / false
datatypes.dart 
void main() {
  int umur = 20;
  double tinggi = 165.5;
  String hobi = 'Coding C++';
  bool isLapar = true;

  // String interpolation ($) buat gabungin variabel
  print('Hobi: $hobi, Lapar: $isLapar');
}

6. Null Safety (PENTING!)

Dart modern itu anti-null. Variabel gak boleh kosong (null) kecuali kamu bolehin secara eksplisit.

  • ? Nullable: String? artinya boleh berisi teks, boleh juga null.
  • ! Bang Operator: "Gue yakin ini gak null!". Kalau ternyata null, aplikasimu crash.
  • late Late: "Nanti gue isi, sumpah". Kalau dipake sebelum diisi, error.
null_safety.dart 
void main() {
  String? alamat; // Boleh null
  print(alamat);  // Output: null

  String kota = 'Bandung'; 
  // kota = null; // Error! Gak boleh null

  // Pakai ! kalau yakin ada isinya
  print(kota.length);
}
